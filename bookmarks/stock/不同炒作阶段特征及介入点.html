    2_不同炒作阶段特征及介入点

    Week.D.Awn 3/7/2024 股票心得

    市场情绪周期轮动：启动--确认--发酵--加速--分歧--反包--龙回头

    # 1.启动

    特点：出现比较有代表性题材换手2连板（可能会带一小范围高潮或没有）。交易机会：打首板

    # 2.确认

    特点：出现比较有代表性题材换手2连板。交易机会:打放量2板

    # 3.发酵

    特点：市场开始出现打破空间作用的连板题材龙头，一般都是3板，会有明显的板块内部个股联动，可能会有涨停潮爆发。交易机会：1.放量后2板的低吸 ，2.打空间3板

    # 4.加速

    特点：市场会出现频繁的缩量涨停，甚至一字板，而且会出现比第三阶段更加汹涌的冲板潮。交易机会：1.新起来去抢主线资金的助攻线空头 ,2.大主线题材下面的补涨接力（卡位）龙头 风险比接近1:1 （交易机会难把握） 3.支流题材龙头介入。

    # 5.分歧

    特点：强连板状态开始减弱，容易出现炸板潮，或者回落导致干的大阴线或者长上影线，也或者是爆量的烂板或者T板等等（明显的亏钱效应）交易机会：1.低吸反包或者分歧后盘中急跌位置或者当天尾盘（具备辨识度龙头票，换手，盘口的强势） 2.新起题材的首板或者确认性的2板（要优选，其成功率较低）

    # 6.反包

    特点：板块龙头最后的活口，市场的分歧非常大，这导致了非常大的成交量，但这个量一般不会超过前一天分歧阶段的成交量。交易机会：1.集合竞价的低开 。2.开盘后的急杀低吸进去。3.涨停板介入。

    # 7.龙回头

    特点：当调整到一定程度会有一个反弹（没有特定规律）交易机会：龙头的低吸（低吸埋伏 )
