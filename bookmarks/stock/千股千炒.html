    # 板块

    # 个股

    艾罗能源: 主营逆变器和储能电池的新股目前动态市盈率才8倍多一点 相比其他普遍15-20倍市盈率的应该是低估的 主要是上市的时间点行情太差了造成的 震有科技: 0308公告的卫星订单还挺大的

    # 基本面

    # 存储方向，周期反转逻辑-模组厂

    简单研究了一下，业绩最先受益弹性最大的是模组厂。因为模组厂从上游晶圆厂进货是合约价，而组装好给下游供货是按现货价，合约价不是实时变化的而现货是，二者涨价存在延迟，所以一旦周期反转，模组厂可以以低价大量进货而以高价卖出，先赚一波差价，再赚一波下游复苏的钱，业绩上最先体现。

    # 固态电池-20240403

    就先梳理一下受益的产业链环节吧 弹性方面 固态电解质＞负极＞正极 固态电解质主要是氧化锆 三祥新材和东方锆业 负极主要增量是硅碳 硅宝科技 元力股份 翔丰华 正极主要增量是高镍 核心是当升科技 还有一个分支就是铲子股 这里面重点看的就是北交所的纳科诺尔 给清陶供应设备 估值才13倍 今晚宁科st发酵了业绩线 业绩好的次新是加分项

    # 经验总结

    • 只跟随，不预判
    • 做在预判中跟随，在跟随中应变，没有预判的跟随是盲从，没有应变的预判是意淫。
    • 指数企稳止跌后的震荡期，往往是牛股的温床。
    • 新方向往往是在分歧中走强，若太一致则很难走出来，所以高潮次日不接力，等分歧。
    • 稀缺性
    • 10日线买点
    • 回踩十日线，理应有个反弹
    • 节奏和仓位的重要性比选股要重要很多
    • 没达到买入条件的票最好不要乱买
    • 情绪会传导，赚钱效应会模仿
    • 赛道如果反弹的话，从阻力最小原则来看，要么炒小要么炒新。炒小的话就参考汽车零部件这一轮的方式，以及充电桩这种小分支，炒新的话阿特斯就可以看看了，目前估值合理，光伏组件全球前五，海外储能系统仅次于特斯拉，且这块业务明年开始放量，未来要占到公司营收的半壁江山。
    • 炒股这种事，说实话只能自渡，你能在韭菜公社和雪球看到的个股的逻辑文基本都是别人自己有持仓才会写文章科普的，不然没有人会这么闲，所以你要有基本的分辨能力。
