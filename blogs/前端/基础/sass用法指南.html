    sass用法指南

    Week.D.Awn 4/13/2022 sass

    # 简介

    学过CSS的人都知道，它不是一种编程语言。

    你可以用它开发网页样式，但是没法用它编程。也就是说，CSS基本上是设计师的工具，不是程序员的工具。在程序员眼里，CSS是一件很麻烦的东西。它没有变量，也没有条件语句，只是一行行单纯的描述，写起来相当费事。

    很自然地，有人就开始为CSS加入编程元素，这被叫做"CSS预处理器"（css preprocessor），包括LESS (opens new window)，SCSS，SASS等。它的基本思想是，用一种专门的编程语言，进行网页样式设计，然后再编译成正常的CSS文件。

    SASS (opens new window)是一种CSS的开发工具，提供了许多便利的写法，大大节省了设计者的时间，使得CSS的开发，变得简单和可维护。
    SASS文件就是普通的文本文件，里面可以直接使用CSS语法。文件后缀名是.scss，意思为Sassy CSS。

    # 1.基本用法

    # 1.1 变量

    SASS允许使用变量，所有变量以$开头。

    $blue : #1875e7;　

div {
　color : $blue;
}
    1
    2
    3
    4
    5

    如果变量需要镶嵌在字符串之中，就必须需要写在#{}之中。

    $side : left;

.rounded {
　　border-#{$side}-radius: 5px;
}
    1
    2
    3
    4
    5

    # 1.2 计算功能

    SASS允许在代码中使用算式：

    body {
　　margin: (14px/2);
　　top: 50px + 100px;
　　right: $var * 10%;
}
    1
    2
    3
    4
    5

    # 1.3 嵌套

    SASS允许选择器嵌套。比如，下面的CSS代码：

    div h1 {
　　color : red;
}
    1
    2
    3

    可以写成：

    div {
　　hi {
　　　　color:red;
　　}
}
    1
    2
    3
    4
    5

    属性也可以嵌套，比如border-color属性，可以写成：

    p {
　　border: {
　　　　color: red;
　　}
}
    1
    2
    3
    4
    5

    注意，border后面必须加上冒号。

    在嵌套的代码块内，可以使用&引用父元素。比如a:hover伪类，可以写成：

    a {
　　&:hover { color: #ffb3ff; }
}
    1
    2
    3

    # 1.4 注释

    SASS共有两种注释风格。

    标准的CSS注释 /* comment */ ，会保留到编译后的文件。

    单行注释 // comment，只保留在SASS源文件中，编译后被省略。

    在/*后面加一个感叹号，表示这是"重要注释"。即使是压缩模式编译，也会保留这行注释，通常可以用于声明版权信息。

    /*!
　　重要注释！
*/
    1
    2
    3

    # 2、代码的重用

    # 2.1 继承

    SASS允许一个选择器，继承另一个选择器。比如，现有class1：

    .class1 {
　　border: 1px solid #ddd;
}
    1
    2
    3

    class2要继承class1，就要使用@extend命令：

    .class2 {
　　@extend .class1;
　　font-size:120%;
}
    1
    2
    3
    4

    # 2.2 Mixin

    Mixin有点像C语言的宏（macro），是可以重用的代码块。

    使用@mixin命令，定义一个代码块。

    @mixin left {
　　float: left;
　　margin-left: 10px;
}
    1
    2
    3
    4

    使用@include命令，调用这个mixin。

    div {
　　@include left;
}
    1
    2
    3

    mixin的强大之处，在于可以指定参数和缺省值。

    @mixin left($value: 10px) {
　　float: left;
　　margin-right: $value;
}
    1
    2
    3
    4

    使用的时候，根据需要加入参数：

    div {
　　@include left(20px);
}
    1
    2
    3

    下面是一个mixin的实例，用来生成浏览器前缀。

    @mixin rounded($vert, $horz, $radius: 10px) {
　　border-#{$vert}-#{$horz}-radius: $radius;
　　-moz-border-radius-#{$vert}#{$horz}: $radius;
　　-webkit-border-#{$vert}-#{$horz}-radius: $radius;
}
    1
    2
    3
    4
    5

    使用的时候，可以像下面这样调用：

    #navbar li { @include rounded(top, left); }

#footer { @include rounded(top, left, 5px); }
    1
    2
    3

    # 2.3 颜色函数

    SASS提供了一些内置的颜色函数，以便生成系列颜色。

    lighten(#cc3, 10%) // #d6d65c
darken(#cc3, 10%) // #a3a329
grayscale(#cc3) // #808080
complement(#cc3) // #33c
    1
    2
    3
    4

    # 2.4 插入文件

    @import命令，用来插入外部文件。

    @import "path/filename.scss";
    1

    如果插入的是.css文件，则等同于css的import命令。

    @import "foo.css";
    1

    # 3、高级用法

    # 3.1 条件语句

    @if可以用来判断：

    p {
　　@if 1 + 1 == 2 { border: 1px solid; }
　　@if 5 < 3 { border: 2px dotted; }
}
    1
    2
    3
    4

    配套的还有@else命令：

    @if lightness($color) > 30% {
　　background-color: #000;
} @else {
　　background-color: #fff;
}
    1
    2
    3
    4
    5

    # 3.2 循环语句

    SASS支持for循环：

    @for $i from 1 to 10 {
　　.border-#{$i} {
　　　　border: #{$i}px solid blue;
　　}
}
    1
    2
    3
    4
    5

    也支持while循环：

    $i: 6;

@while $i > 0 {
　　.item-#{$i} { width: 2em * $i; }
　　$i: $i - 2;
}
    1
    2
    3
    4
    5
    6

    each命令，作用与for类似：

    @each $member in a, b, c, d {
　　.#{$member} {
　　　　background-image: url("/image/#{$member}.jpg");
　　}
}
    1
    2
    3
    4
    5

    # 3.3 自定义函数

    SASS允许用户编写自己的函数。

    @function double($n) {
　　@return $n * 2;
}

#sidebar {
　　width: double(5px);
}
