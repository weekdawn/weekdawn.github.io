    k8s基本命令与pod管理

    Week.D.Awn 1/17/2022 k8spod

    工作中常用到的 k8s 命令备份，方便查看。

    # 查看指定命名空间下的所有节点

    kubectl get pod -n weekdawn
    1

    # 进入某一节点

    kubectl exec -it test-pod-statefulset-1 /bin/bash -n weekdawn
    1

    # 删除某一节点

    kubectl delete test-pod-statefulset-1 -n weekdawn
    1

    # 查看pod暴露的端口号

    kubectl get svc -n weekdawn
    1

    # 移动文件至pod内部

    移到文件到容器中pod的pvc目录下后重启pod，pvc目录中的文件不会丢失

    kubectl -n weekdawn cp ./aaa test-pod-statefulset-1:/bbb/sdf
    1

    # 修改ConfigMap

    同理修改：pod、svc、statefulset、cm（cm代表ConfigMap）

    # 查看ConfigMap
kubectl get cm -n obp
# 编辑
kubectl edit cm test-pod-config -n weekdawn
# 删除
kubectl delete cm test-pod-config -n weekdawn
